# Full Package Health Check

## 1. Update Julia itself

# Terminal command line:
juliaup status         # check what versions you have
juliaup update          # update juliaup itself + all installed Julia channels
juliaup add release     # ensure latest stable is installed
#juliaup add lts         # ensure latest LTS (1.10.x) is installed
juliaup status          # confirm what versions you have

## 2. Ensure you are in dev/ branch (not main/ branch) and have pulled latest changes
git checkout dev
git pull origin dev

## 3. Update All Dependencies
# In Julia REPL:
using Pkg
Pkg.activate(".")
Pkg.update()          # update all packages to latest compatible versions
Pkg.status()          # check status of all packages

## 4. Check for Package Issues
Pkg.resolve()         # resolve any dependency issues
Pkg.precompile()      # precompile all packages

## 5. Run Tests
Pkg.test()            # run all tests defined in the package
